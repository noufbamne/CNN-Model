# 🧠 CNN Model
> Convolutional Neural Network implementation with training, testing, and preprocessing workflows.

---

## 🚀 Overview
This repository contains the implementation of a Convolutional Neural Network (CNN) model, including training, testing, and data preprocessing pipelines.

It focuses on understanding the complete workflow of building and evaluating deep learning models.

---

## 🧠 Features
- 🤖 CNN model training  
- 📊 Exploratory Data Analysis (EDA)  
- 🔍 Testing with and without preprocessing  
- ⚡ Model evaluation scripts  

---

## 🛠 Tech Stack
- Python  
- TensorFlow / Keras  
- NumPy, Pandas  

---

## 📂 Structure
```bash
CNN-Model/
│── Model_2.ipynb
│── Model_2.py
│── EDA_2.py
│── Model_Test_*.py
│── metadata_2.csv
│── label_encoder_2.pkl

```

---

## ⚙️ Workflow
1. Perform data analysis (EDA)  
2. Train CNN model  
3. Apply preprocessing techniques  
4. Evaluate model performance  
5. Compare results with/without preprocessing  

---
